# 🚀 Self-Host Discord Bot



## 📖 Описание проекта

Этот Discord-бот разработан для того, чтобы пользователи могли легко запускать и настраивать его на своих устройствах. Бот поддерживает базовые функции, а также возможность проигрывания аудио через голосовые каналы. Проект идеально подходит для тех, кто хочет создать своего бота без необходимости глубоких знаний в программировании. [Список команд](commandlist.md)

## 🎯 Предварительные требования

- **Python 3.13**
- Токен Бота от [портала разработчиков Discord](https://discord.com/developers/applications)
- Для аудио-функций: **FFmpeg**

---

## ⚙️ Настройка конфигурации

После успешного запуска бота в терминале появятся ссылки для доступа к веб-интерфейсу. Перейдите по одной из них в вашем браузере.

В веб-интерфейсе вы сможете настроить конфигурацию бота. Вам потребуется:

- **Токен бота**: Получите его на [портале разработчиков Discord](https://discord.com/developers/applications)
- **Префикс команд**: Установите префикс, который будет использоваться для вызова команд бота
- **Каналы и роли**: Настройте, какие каналы и роли будет использовать бот

> **⚠️ Важно**: 
> - При изменении токена бота в конфиге необходимо **перезапустить веб-интерфейс**
> - Иконки для эмбедов принимают только **URL** на картинку
> - Цвета оформления эмбедов обязательно вводить **hex-код**

## 🎵 Настройка аудио

Для работы бота с аудио необходима программа **FFmpeg**:

- **Windows**: [Инструкция в руководстве по установке](instruction/win_install.md)
- **Ubuntu**: [Инструкция в руководстве по установке](instruction/ubuntu_install.md)

> Для работы треков с **Яндекс музыки** понадобится экспортировать её куки (в **Netscape** формате), используя любое расширение с браузера для экспорта куков. Их нужно вставить в `cookie.txt` в корневой папке проекта. К сожалению без этого никуда =(

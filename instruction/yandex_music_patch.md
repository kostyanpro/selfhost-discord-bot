# 🎶 Насройка бота для работы с яндекс музыкой

## 1. Создание куки файла

Для работы треков с **Яндекс музыки** понадобится экспортировать её куки, используя любое расширение с браузера для экспорта куков ([например это](https://chromewebstore.google.com/detail/cookie-editor/hlkenndednhfkekhgcdicdfddnkalmdm?hl=ru&utm_source=ext_sidebar)). Их нужно вставить в `cookie.txt` в корневой папке проекта. К сожалению без этого никуда =(

> **💡 Важно:** Куки должны быть в *Netscape* формате

## 2. Патч yt_dlp

Запустить `patch.bat` в папке `yandexMusicPatch`.
> Почему-то yt_dlp обращается к яндекс музыке по http, из-за этого бот получает ошибку *403*. Патч просто меняет пару строк в библиотеке и исправляет это
